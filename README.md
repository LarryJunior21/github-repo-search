# GitHub Repo Search

A clean, responsive GitHub repository search app built with Next.js and TypeScript. Search any GitHub user or organisation and browse their repositories instantly.

## Features

- Search GitHub users and organisations by username
- Browse all public repositories with key details
- Fully responsive, works on mobile and desktop
- Fast and lightweight, built on the GitHub public API

## Tech Stack

- **Next.js** with TypeScript
- **GitHub REST API**
- Responsive layout with CSS/Tailwind

## Getting Started

```bash
# Clone the repo
git clone https://github.com/LarryJunior21/github-repo-search.git
cd github-repo-search

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
